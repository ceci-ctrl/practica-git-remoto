\# Mi Proyecto de Prácticas de Git y GitHub



Este es un repositorio educativo estructurado y desarrollado paso a paso para aprender a dominar el control de versiones local y remoto con Git.



\## Funcionalidades principales

\* Registro y auditoría del historial completo de desarrollo.

\* Creación de flujos de trabajo aislados mediante ramas (\*branching\*).

\* Resolución de conflictos de fusión de manera manual y segura.

\* Sincronización en tiempo real con servidores en la nube en GitHub.



\## Tabla de Comandos Git Básicos



| Comando | Acción | Propósito del comando |

| :--- | :--- | :--- |

| `git status` | Consulta | Comprueba el estado actual de los archivos |

| `git add .` | Preparación | Añade las modificaciones al escenario (\*stage\*) |

| `git commit -m "msj"` | Confirmación | Registra una instantánea fija en el historial |

| `git pull` | Sincronización | Descarga y fusiona los cambios del servidor remoto |
